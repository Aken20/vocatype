## What This Project Is

Local Windows dictation app. Press Ctrl+Shift+. to start dictating anywhere. AI-powered voice-to-text with optional LLM cleanup. Built for speed and privacy.

## Stack

| Layer | Technology |
|-------|-----------|
| Desktop Shell | Tauri 2 (Rust) |
| Frontend | React + Zustand + Tailwind |
| Backend | Python 3.11 + FastAPI (localhost sidecar) |
| STT Engine | faster-whisper small (CUDA via nvidia-cublas-cu12) |
| Audio Capture | WASAPI (Windows) |
| LLM Cleanup | Optional — LM Studio Llama-3.2-3B |
| Monitoring | Tkinter pill window (auto-starts) |

## Key Conventions

- **Hotkey:** Right Ctrl (0xA3) + Right Shift (0xA1) → `SetWindowsHookEx` for global detection. Must match BOTH left and right modifier variants.
- **Python backend runs as sidecar** — started by Tauri, communicates via HTTP/WS on localhost
- **CUDA workaround:** nvidia-cublas-cu12 for CUDA 13.2 compatibility with faster-whisper
